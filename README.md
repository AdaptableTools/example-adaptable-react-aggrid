# example-adaptable-react-aggrid

## React Wrapper Example
This is a very basic example of how to install and get started with the Adaptable React Wrapper.

It uses v.7 of AdapTable and v.23 of ag-Grid

For full information on how to use the AdapTable React Wrapper see the package [Read Me](https://github.com/AdaptableTools/adaptable/blob/master/packages/adaptable-react-aggrid/README.md).

### Installation

NOTE: In order to be able to run `npm install`, you need to be logged into our private NPM registry - follow the instructions in the [Adaptable React Wrapper Read Me](https://github.com/AdaptableTools/adaptable/blob/master/packages/adaptable-react-aggrid/README.md)

> If you do not have an Adpatable Login please contact support@adaptabletools.com

Run `npm install` (or `yarn`), depending on what tool you're using.

### Running for development

Execute the following command

```sh
$ npm run start
```
